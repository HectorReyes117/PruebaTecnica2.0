# Prueba_Tecnica

### Como correr el proyecto?

```


Para correr el proyecto debe digirse al appsetting.json e introducir su Servidor de Sql Server donde dice Server.

Luego dirigirse a visual studio comunity a Herramientas > Administrador de paquetes NuGet > Consola del Administrador de Paquetes. 
En dicha consola debera escribir el comando update-database y para aplicar la migracion y ya tendria listo el proyecto para correr.

```
